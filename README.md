# Powerbi Sales Dashboard
---

## 📝 Giới thiệu dự án
Dự án này là một giải pháp Dashboard toàn diện nhằm phân tích **hiệu suất kinh doanh**, **hành vi khách hàng** và **hiệu quả sản phẩm** dựa trên dữ liệu bán hàng thực tế.  

Hệ thống cung cấp cái nhìn sâu sắc từ tổng quan đến chi tiết, giúp nhà quản lý đưa ra các quyết định kinh doanh dựa trên dữ liệu (*data-driven decisions*).

---

## 🚀 Thành phần dự án
Dự án bao gồm 

- 📊 **Power BI Report**: Hệ thống báo cáo tương tác mạnh mẽ phục vụ phân tích chuyên sâu  
- 💻 **Dashboard UI**: Giao diện Dashboard hiện đại, trực quan hóa các chỉ số 


## 🔑 Các chỉ số quan trọng (KPIs)

### 1. Phân tích Bán hàng (Sales Product)
* **Tổng quan KPI:** Theo dõi doanh thu thực tế (Revenue), Tổng đơn hàng (Total Orders) và Tỷ lệ hoàn thành (Completion Rate).
* **Trạng thái đơn hàng:** Phân tích tỷ lệ đơn hàng Hoàn thành vs Bị hủy (66% Success rate).
* **Vận chuyển:** Đánh giá hiệu quả các phương thức giao hàng (Overnight, Standard, Express).

### 2. Phân tích Sản phẩm (Product Insights)
* **Sản lượng tiêu thụ:** Biểu đồ cột chồng theo dõi biến động 5 dòng sản phẩm chủ lực (Office Chair, Office Desk, Storage Cabinet, Bookshelf, Desk Lamp) xuyên suốt 12 tháng.
* **Ma trận hiệu suất:** Bảng chi tiết doanh thu hàng tháng cho từng loại sản phẩm.
* **Top Revenue:** Xác định **Office Chair** là sản phẩm mang lại dòng tiền lớn nhất.

### 3. Phân tích Khách hàng (Customer Demographics)
* **Nhân khẩu học:** Phân nhóm khách hàng theo độ tuổi (Nhóm 26-60 tuổi chiếm tỷ trọng ~63%).
* **Lòng trung thành:** Phân tách rõ rệt giữa Khách hàng mới (69%) và Khách hàng cũ (31%).
* **Xếp hạng VIP:** Bảng vinh danh **Top Loyal Customers** dựa trên tổng chi tiêu cộng dồn (Aggregated Spend).
